# Change Log

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2025-11-19

### Added

- Support for **Vercel** (files and folders).

## [1.0.0] - 2025-11-15

### Added

- **Web Ecosystem:**
  - Support for **React** and **Svelte**.
  - Support for **Vite** configuration files.
  - Icons for **SASS** and **Less** styles.
  - Icons for **Prettier**, **ESLint**, and **Babel** configurations.
- **Runtimes & Package Managers:**
  - Comprehensive support for **Node.js**, **Deno**, **Bun**, and **PNPM** (including lockfiles and configs).
  - **Python** (Pip/Pipfile) and **Ruby** (Gemfile) support.
  - **Rust** (Cargo) support.
- **Languages & Data:**
  - Icons for **Lua** and **Prisma** schemas.
  - Support for **YAML**, **TOML**, **XML**, and **Env** configuration files.
  - Icons for **Font** files.
- **Tools & Shell:**
  - Expanded **Docker** support (Dockerfiles and `.dockerignore`).
  - **Terminal** and **Shell** script icons (PowerShell, Bash, Zsh).
  - **VS Code** specific icons (extensions, ignore files, `.vscode` folder).
  - Icons for **License**, `robots.txt`, and `GEMINI.md` files.
- **Development:**
  - `generate-samples` script to create test files for all mapped icons.

### Changed

- Adjusted Node.js folder icon proportions and colors.
- Refactored filenames for C# and C++ (sanitized to `csharp` and `cpp`) for better compatibility.

## [0.0.0] - 2025-09-21

### Added

- Initial icons done for the following:
  - **Files:** Aseprite, Blender, C, C++, C#, CSS, Database, Docker, GDScript, Git, Godot Scene, HTML, Image, JavaScript, JSON, Markdown, Model, Ruby, Rust, Sound, SVG, Text, TypeScript (including test and config files), Video, Vite, Vue, and Zip.
  - **Folders:** Default, Assets, Build, Data, Dist, Extensions, Models, Music, Python, Sound, and VSCode.
